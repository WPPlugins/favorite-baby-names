== Description ==

Adds a customizeable widget which displays the latest favorite baby names in Germany by [vorname.com](http://www.vorname.com/)

*Feature List*

* customizable widget
* displays a user-defined number of names

== Installation ==

1. Extract `favorite-baby-names.zip`.
2. Upload the `favorite-baby-names`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.

*German*
a. Entpacke `favorite-baby-names.zip`.
b. Lade den `favorite-baby-names`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
c. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
d. F&uuml;ge das Widget unter Design->Widgets deiner Sidebar hinzu und bearbeite die Widget-Optionen.
